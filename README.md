# Animaid

A GUI toolkit for animating Python data structures.

## Installation

```bash
pip install animaid
```

## Development

```bash
# Clone the repository
git clone https://github.com/jdrumgoole/animaid.git
cd animaid

# Install with development dependencies
uv pip install -e ".[dev,docs]"

# Run tests
invoke test

# Run all checks
invoke check

# Build documentation
invoke docs
```
